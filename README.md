# static-blog

私のブログです。  
Astro + MDX + JS Plugins で構成されていて、 GitHub Actions でビルドし GitHub Pages で公開されます。  
<u>*ブログの内容や画像などの資料は [CC BY-NC-SA 4.0](https://creativecommons.org/licenses/by-nc-sa/4.0/) で公開されています。*</u>

## 編集者向け

- forkしたあとのセットアップ手順 [README.dev.md](./README.dev.md)
  - ブログタイトルの変更や既存の記事の削除、画像の差し替えなどを行ってから使用してください。
- マークダウン記法 [README.writer.md](./README.writer.md)
  - このテンプレートにはいくつかの特殊記法に対応するためのJSプラグインが含まれています。
